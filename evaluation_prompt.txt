I have built a traffic flow forecasting framework as part of my research project. It implements four Spatio-Temporal Graph Neural Network models — AMC-DSTGNN, CADGT, CAMT-GATformer, and ST-ACENet — trained and evaluated on the METR-LA dataset (207 sensors, Los Angeles).

The framework makes three core technical claims:
1. Dynamic Spatial Dependency Learning — Graph structure changes per iteration based on traffic state, blending static road geometry with learned dependencies.
2. Multi-Context Integration — Models integrate external factors (temperature, rainfall, wind speed, visibility, holidays, time-of-day, day-of-week) alongside traffic speed.
3. Unified Short-Term and Long-Term Temporal Modeling — A single model handles 5-minute to 60-minute forecasting in one forward pass (12-step horizon).

Implementation details:
- Unified train.py and test.py that train/test all four models with identical data pipeline
- 10-feature input tensor (traffic + weather + holidays + temporal encodings)
- AdamW optimizer, early stopping (patience=8), ReduceLROnPlateau scheduler
- Mixed precision (AMP) training with gradient clipping
